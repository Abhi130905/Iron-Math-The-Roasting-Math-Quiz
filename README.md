# Iron Math 🤖🧮

Welcome to **Iron Math**, a snarky, Avengers-themed web application that tests your basic arithmetic skills and isn't afraid to hurt your feelings. If you get a question wrong, prepare to be roasted! 

## 🌟 Features

- **Dynamic Math Questions**: Challenges you with randomly generated addition, subtraction, and multiplication problems.
- **Witty Feedback System**: 
  - Get a question wrong? The app will roast you with insults that even Tony Stark would approve of.
  - Get it right? You'll receive some mild, slightly condescending praise.
- **Progress Tracking**: A sleek progress bar and score tracker keeps you updated throughout the 10-question quiz.
- **Final Verdict**: Receive a personalized verdict at the end of the quiz based on your score (ranging from "You are Iron Man" to "I'm revoking your Avengers access card").
- **Responsive UI**: Built with a beautiful "glassmorphism" design and interactive shake animations when you make mistakes!

## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Build Tool**: [Vite](https://vitejs.dev/) for blazing fast development and bundling.
- **Typography**: Google Fonts (Outfit)

## 🛠️ Getting Started

### Prerequisites

You will need [Node.js](https://nodejs.org/) installed on your computer to run the development server.

### Installation & Running Locally

1. **Clone the repository** (if applicable) or download the source code:
   ```bash
   git clone <your-repository-url>
   cd "Iron math"
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Play the quiz**: Open your browser and navigate to `http://localhost:5173/` to start getting roasted!

## 🏗️ Project Structure

- `index.html`: The main entry point containing the layout and UI elements.
- `src/main.js`: Contains all the application logic, question generation, and the snarky roasts.
- `src/style.css`: All the styling, animations, and responsive layout code.
- `package.json`: Manages the project dependencies and scripts.
